// why you can let it work

Built to be trusted with your notes.

The structure that makes a vault legible to you is the same structure that makes it queryable by agents. Silica protects it.

Local-first

[ fs ]

Plain Markdown on your disk. No daemon, no server, no cloud round-trip. Obsidian-compatible, but it doesn't need Obsidian to run.

Reversible

[ journal ]

Snapshots before, gates after, undo for everything. Uncertain edits go to a review queue, not your vault. A regression never lands silently.

Human-first

[ vault ]

The vault stays legible to you. Embeddings and co-occurrence are derived indexes — rebuildable, never the source of truth. No parallel store.

// install — python 3.11+, apache-2.0

Point it at a vault.

Clone, install editable, run the wizard. It writes your .env and ends with a diagnostic so you know it's healthy.

$ git clone https://github.com/kiycoh/silica-agent.git
$ cd silica-agent
$ uv pip install -e .
$ uv run silica init     # wizard: vault, backend, models
$ uv run silica          # start the REPL
